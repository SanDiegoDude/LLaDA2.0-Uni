# =============================================================================
# Two-step install (see INSTALL.md for the full story):
#
#   1. Install torch FIRST from the PyTorch index that matches your CUDA
#      driver. Do NOT rely on the PyPI default — it pulls torch 2.11+cu130
#      which needs a CUDA 13 runtime and also has no prebuilt flash-attn.
#
#      Blackwell / DGX (CUDA 13 driver):
#        pip install torch --index-url https://download.pytorch.org/whl/cu130
#      RTX 40-series / CUDA 12.8 driver:
#        pip install torch==2.7.1 torchvision --index-url https://download.pytorch.org/whl/cu128
#      CUDA 12.4 driver (older):
#        pip install torch==2.5.1 torchvision --index-url https://download.pytorch.org/whl/cu124
#
#   2. Then install the rest:   pip install -r requirements.txt
#   3. Finally flash-attn:      pip install --no-build-isolation flash-attn==2.8.3
#
# transformers is hard-pinned because 5.x broke the custom trust_remote_code
# loader path used by the LLaDA-2.0-Uni modeling file.
# =============================================================================
